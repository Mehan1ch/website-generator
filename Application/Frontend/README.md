# Frontend

A React frontend for the application, built with Vite and TypeScript.

---

## Running the frontend in development mode

- Prerequisites:
    - Node.js and npm or yarn installed.
    - Backend server running and accessible.
    - OpenAPI specification file generated by the backend (e.g., `api/v1.yaml`).

- Steps:
    1. Copy `.env.example` to `.env` and set the variables.
    2. Generate the OpenAPI client using openapi-ts with
       `npx openapi-typescript ./path/to/api/v1.yaml -o ./src/lib/api/v1.d.ts`
    3. Start the frontend container with Docker Compose:
       ```bash
       docker-compose -f docker-compose.yaml up
       ```
    4. Access the frontend at `http://localhost:3000`.
